# Koicosmos Home

Personal site for `koicosmos.com`, built with Astro and Markdown.

## Structure

- `src/content/blog`: Markdown posts
- `src/content/projects`: project entries
- `src/pages/about`: searchable resume/profile page
- `src/pages/admin`: editor entry for Pages CMS

## Deployment

Deploy with Vercel:

- Framework preset: Astro
- Build command: `npm run build`
- Output directory: `dist`
- Production domain: `koicosmos.com`

Keep Sensus on its own subdomain:

- `sensus.koicosmos.com`
